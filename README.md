# AI Learning English 🦉

一个给六年级小朋友用的**背单词网页**：打开就能学、能听发音、能做测验。

## 怎么打开

直接双击 `index.html`，用浏览器（Chrome / Edge）打开即可。

## 功能

- 📖 **学单词**：卡片显示英文，点一下看中文，点 🔊 听发音
- ✏️ **做测验**：看单词选中文意思，答对得星星 ⭐
- 💾 自动记住得分

## 怎么加新单词

打开 `index.html`，找到 `const WORDS = [` 那一段，照着格式增加即可：

```js
{ en: "hello", cn: "你好" },
```
